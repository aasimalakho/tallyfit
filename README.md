# Tallyfit

A scorecard for people who bid for a living. Upload your CV and paste a job
post — Tallyfit scores the fit, shows what's aligned and what's missing, and
hands you a ready-to-use proposal opening line. No AI model, no API key —
pure rule-based skill matching, fast and predictable.
